# devanandersen.github.io

Personal website for [devanandersen.com](https://devanandersen.com) — a pixel-art endless runner built with Preact and Vite.

## Stack

- [Preact](https://preactjs.com/) — UI
- [Vite](https://vitejs.dev/) — build tool
- GitHub Actions — CI/CD to GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Deployments are handled automatically via GitHub Actions on push to `master`.
